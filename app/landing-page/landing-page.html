Cryptocurrency Mining Calculator

A free browser-based tool providing predictive block-solving statistics and cost projection for cryptocurrency mining.

Statistical Forecasting

This gives estimations of potential profits and loss for cryptocurrency mining.

It provides various advanced statistics for miners given a input variables (eg, hashrate, difficulty, power costs etc).

Capable of difficulty prediction using real-time blockchain statistics using some cool math.

Launch Statistical Forecaster

Miner Performance Analysis

Enter your miners Ethereum address and see how it is performing compared to its statistical expectation.

This is very useful for detecting issues with your miner as it gives a clear statistical measure (standard deviation) to determine its operational performance.

This is designed for solo-mining addresses only. Pool miners have no use for this feature.

FYI

At this stage we're mainly focused on Ethereum, however you'll find you can apply aspects of the forecasting tool to most cryptocurrencies. This project is still under active development, there will be bugs.

We used Chrome to develop this. You can probably use another browser but testing for cross-compatibility is boring and we didn't do it. If you're running an up-to-date browser (that isn't IE) and you have an issue, raise it on our GitHub.

We don't collect any data, we promise. You can even check our open-source code.

We use some external, public APIs to provide us with data. We cannot assure the security or accuracy of these services.
